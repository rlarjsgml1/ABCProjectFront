## 7. 주요 버그 수정 / 개선

- 도서 목록/검색 결과 API 실패 시 가짜 데이터로 화면을 가리지 않도록 수정
- 마이페이지 사이드바 이동 시 스크롤 튕김 현상 수정
- 포인트/쿠폰 필드 정합화 및 마이페이지 스크롤 유지
- 독서 통계 API 응답 매핑 오류 수정 (완독 수 추이로 라벨 정정), 통계 화면 크래시 수정
- 추천 도서 API 응답 파싱 오류 수정
- 성별 값을 백엔드 규칙(`MALE`/`FEMALE`/`NONE`)으로 통일
- 약관 동의 체크박스가 취소 후에도 체크된 것처럼 남는 버그 수정
- 회원가입 검증 실패 시 해당 입력창으로 스크롤/포커스 이동
- 컬렉션 시리즈 필드 비활성화 및 실제 도서 검색 연동 (Admin)
- 도서 없는 컬렉션에 가려 시리즈 미리보기가 안 뜨는 문제 수정 (홈)
- 헤더가 스크롤 시 사라지는 문제 수정 (`position: sticky` + `overflow-x: hidden → clip`)
- 버튼 클릭 시 텍스트 커서 노출 방지 등 다수 UI 디테일 수정
